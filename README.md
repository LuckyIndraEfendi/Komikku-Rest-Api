# Komikku API
Restful API Manga bahasa Indonesia built with ❤️ and Go

# Usage
1. Clone this repository
    ```bash
    git clone https://github.com/LuckyIndraEfendi/Komikku-Rest-Api.git
    ```
2. Init module
    ```
   go mod init komikku-api
   ```

3. Installing dependencies
    ```
   go mod download
   ```
4. Build binary file
   ```
   make build
   ```
5. Run program
   ```
   make start
   ```
# Using Docker Compose
#### Build image and run container
   ```
   make up
   ```

#### Remove image and stop container
   ```
   make down
   ```

# API Output Example

## Get All Comic

#### Get Comic List

   ```
   https://komikku-rest-api-production.up.railway.app/api/comic/list
   https://komikku-rest-api-production.up.railway.app/api/comic/list?filter=manga
   https://komikku-rest-api-production.up.railway.app/api/comic/list?filter=manhwa
   https://komikku-rest-api-production.up.railway.app/api/comic/list?filter=manhua
   ```

#### Get Popular Comic
   ```
   https://komikku-rest-api-production.up.railway.app/api/comic/popular/page/1
   ```

#### Get Recommended Comic
   ```
   https://komikku-rest-api-production.up.railway.app/api/comic/recommended/page/1
   ```

#### Get Newest Comic
   ```
   https://komikku-rest-api-production.up.railway.app/api/comic/newest/page/1
   ```
